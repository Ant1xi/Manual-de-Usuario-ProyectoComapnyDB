Ejercicio 1 - Alta de Nuevo Empleado

📌 ¿Para qué sirve?

Esta funcionalidad permite registrar un nuevo empleado en el sistema de la empresa. El usuario puede introducir los datos básicos del trabajador y guardarlos fácilmente.

🧭 ¿Cómo funciona?

  1. El usuario abre el programa y hace clic en el botón "Alta de Empleado".
  2. Se abre una ventana con un formulario.
  3. Se rellenan los campos con los datos del nuevo empleado.
  4. Al pulsar el botón “Guardar”, el sistema guarda la información.

📋 ¿Qué datos hay que introducir?

  • Nombre
  • Apellidos
  • Email
  • Teléfono
  • Fecha de contratación
  • Cargo
  • Jefe (opcional, desde una lista desplegable)

⚠️ Si hay errores

Si falta algún dato obligatorio, el programa mostrará un aviso pidiendo completar los campos.

✅ Resultado

Si todo está correctamente rellenado, aparece un mensaje confirmando que el empleado ha sido añadido correctamente.

🎯 Conclusión

Con este formulario, se facilita la gestión del personal permitiendo añadir nuevos empleados de manera rápida, sencilla y segura.
