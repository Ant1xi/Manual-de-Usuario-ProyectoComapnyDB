Ejercicio 2 - Alta de Nuevo Almacén

📌 ¿Para qué sirve?

Este apartado permite dar de alta un nuevo almacén en el sistema, indicando su ubicación a través de una selección guiada por región, país y ciudad.

🧭 ¿Cómo se utiliza?

  1. Desde el menú principal, se hace clic en el botón "Alta de Almacén".
  2. Se abre una ventana con varios campos y listas desplegables.
  3. Primero se elige la región.
  4. Después, se selecciona un país dentro de esa región.
  5. Luego, se escoge la ubicación concreta dentro del país.
  6. Por último, se escribe el nombre del nuevo almacén y se pulsa el botón "Crear nuevo almacén".

🧾 ¿Qué datos hay que introducir?

  • Región: Se elige de una lista desplegable.
  • País: Aparece según la región seleccionada.
  • Ubicación: Se selecciona según el país elegido.
  • Nombre del almacén: Escrito por el usuario.

⚠️ ¿Qué pasa si falta algo?

Si no se ha seleccionado región, país o ubicación, o si el nombre del almacén está vacío, el sistema mostrará un aviso indicando que hay que completar todos los campos.

✅ ¿Qué pasa si todo va bien?

Si todos los campos están correctamente rellenados, el sistema guarda el nuevo almacén y muestra un mensaje de confirmación. Después se cierra la ventana.

🎯 Resultado final

Una vez creado, el almacén queda registrado en el sistema con la ubicación seleccionada. Esta funcionalidad permite mantener organizada la gestión de los almacenes de forma clara y sencilla.
